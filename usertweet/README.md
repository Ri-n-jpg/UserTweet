# UserTweet Django App

A simple Django web application for creating, editing, and deleting tweets with user authentication.

---

## Features

- User registration with email, username, and password.
- User login and logout functionality.
- Create, edit, and delete tweets (text + optional photo).
- Restrict edit/delete actions to the tweet owner.
- File uploads handled securely (media folder excluded from GitHub).
- Responsive UI with Bootstrap.

---

## Tech Stack

- Django 6.0.3
- Python 3.14
- SQLite (default, can be changed)
- Bootstrap for frontend styling
